왜 Gemini를 선택했나?

💰 완전 무료 API

Google AI Studio 무료 제공
하루 100만 요청까지 무료
GPT-4 대비 125배 저렴
(Flash-Lite: $0.02/100만 토큰)

🚀 뛰어난 성능

Gemini 2.5 Flash: 빠른 응답 속도
100만 토큰 컨텍스트 윈도우
긴 문서와 대화 처리 가능

🎯 멀티모달 AI

텍스트, 이미지, 오디오 통합 처리
Google Search 연동 가능
Deep Research로 자동 리포트 생성

⚡ 공식 CLI 도구

오픈소스 Gemini CLI 제공
ReAct 루프로 복잡한 작업 자동화
MCP로 도구 확장 가능

🎓 이 수업에서 Gemini를 선택한 이유

  • 무료 API: 학생들이 비용 걱정 없이 실습 (250~500 요청/일)
  • 공식 CLI 도구: 설치와 사용이 간단 (npm 한 줄)
  • 적절한 성능: 교육용으로 충분한 코딩 성능 (63.8%)
  • 빠른 응답: 실습 대기 시간 최소화 (250+ TPS)
